矩阵运算
===================================
##功能说明
这是一个有关于矩阵的程序，提供了一个关于矩阵的数据结构和矩阵相关的运算的函数

##configuration文件中的参数说明：
###input表示输入信息来源
	如果为00，表示从输入设备中获得值
	如果为01，表示从source.txt文件中获得值
###output表示结果输出方式
	如果为10，表示为仅输出到输出设备
	如果为01，表示为仅输出到output.txt文件中
	如果为11，表示同时输出到输出设备和output.txt文件中

##source.txt文件中的数值的格式说明
	所有数字可以写在同一排，也可以按照矩阵的排列方式分成多排，每个数字之间用空格，换行，或者制表符隔开

##RowReduce说明
	即行化简，为保证计算精度，只计算到三角形式，剩余部分需要使用者自行计算
